This repository contains a collection of practice notebooks focused on core Python concepts and numerical computing using NumPy. The exercises are organized into six days, each covering different topics to strengthen programming and data analysis skills.

## Topics Covered

### Day 1 – Python Data Structures

* Lists, Tuples, Sets, and Dictionaries
* Properties: ordered/unordered, mutable/immutable, unique values
* Data structure conversions (list → set, tuple → list, etc.)
* Dictionary example for storing student names and grades

### Day 2 – String Manipulation

* String slicing (first word, last word, middle section)
* String formatting techniques
* String methods: `upper()`, `lower()`, `replace()`, `split()`, `join()`
* Sentence and paragraph manipulation

### Day 3 – Introduction to NumPy Arrays

* Understanding arrays and their advantages over Python lists
* Creating NumPy arrays
* Exploring array attributes: shape, size, and data type
* Performance comparison between lists and arrays

### Day 4 – Array Operations

* Array indexing and slicing
* Extracting rows, columns, and sub-arrays
* Reshaping arrays (1D → 2D → 3D)
* Combining arrays using stacking and concatenation

### Day 5 – Math & Statistics with NumPy

* Element-wise operations (addition, subtraction, multiplication)
* Statistical functions: sum, mean, median, max, min, standard deviation
* Aggregation across rows and columns in matrices
* Dataset analysis example (exam scores)

### Day 6 – Practice Session

* Matrix operations: addition, subtraction, multiplication
* Generating random numbers and matrices
* Simulating real-world scenarios such as dice rolls and random sampling
